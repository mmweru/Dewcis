## Overview

A complete file archiving system that:
1. Archives files from Linux group members' home directories
2. Stores every event in PostgreSQL progressively
3. Provides REST API to query archive history
4. Includes a dashboard that auto-refreshes every 10 seconds

- Docker Desktop installed and running
- Docker Compose v2+
- Git (optional, for cloning)
